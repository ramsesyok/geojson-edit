# GeoJSON Edit (offline)

オフライン / エアギャップ環境で動作する、内蔵マップ付きの GeoJSON 編集用 VSCode 拡張機能です。
地図データ(`world.pmtiles`)を同梱し、外部ネットワークや HTTP サーバなしで地図を表示・編集できます。

## 特長

- **完全オフライン**: 地図データを拡張機能に同梱。CDN・HTTP サーバ不要(WebView から `asWebviewUri` でメモリ読み込み)。
- **カスタムエディタ**: `.geojson` を開くと地図エディタで表示。通常のテキストファイルのままなので、保存・Undo/Redo・差分表示は VSCode 標準どおり。
- **作図ツール**: 点 (Point) / 線 (LineString) / 面 (Polygon) / 円 (Circle) / 削除。
- **編集**: クリックで選択(ハイライト＋頂点●)、頂点編集・平行移動・円のリサイズ、スナップ対応。
- **座標編集パネル**: 選択地物の緯度経度を数値で直接編集(点・円・線・面)。
- **「更新」ボタン方式**: 選択中の編集は下書きで、「更新」を押すまでファイルに保存しない。
- **プロパティ編集**: ワークスペース設定で定義したフィールド(string / number / boolean / enum / color)を選択地物に入力。
- **name / color**: 組み込みフィールドとして常に編集可能。`name` を地図上にラベル表示、`color`(HEX)で地物を着色。

---

## はじめに

1. `.geojson` ファイルを開くと、自動で **GeoJSON Map Editor** で開きます。
   - 新規作成: コマンドパレット(`Ctrl+Shift+P`)から `GeoJSON Edit: 新規 GeoJSON を作成`。
2. 左上の**ツールバー**で操作モードを選び、地図上で作図・編集します。
3. `Ctrl+S` で保存(通常の GeoJSON テキストファイルとして出力)。Undo/Redo も VSCode 標準どおり。

初回表示時は、データが含まれていればその範囲へ自動でズームします。

---

## ツールバー

左上のツールバーで操作モードを切り替えます。

| ボタン | モード | 用途 |
| --- | --- | --- |
| ✏️ 編集 | 選択・編集 | 既存の地物を選択して頂点・位置・座標・プロパティを編集 |
| • 点 | Point 作図 | クリックした位置に点を作成 |
| ╱ 線 | LineString 作図 | クリックで頂点を追加、**ダブルクリックで確定** |
| ▰ 面 | Polygon 作図 | クリックで頂点を追加、**ダブルクリックで確定** |
| ◯ 円 | Circle 作図 | クリックで中心 →もう一度クリック(またはドラッグ)で半径を確定 |
| 🗑 削除 | 削除 | クリックした地物を削除 |

> 作図・編集中は、既存の頂点に**スナップ**します。

---

## 図形を描く(作図ツール)

1. ツールバーで **点 / 線 / 面 / 円** のいずれかを選びます。
2. 地図上で作図します。
   - **点**: 置きたい位置をクリック。
   - **線 / 面**: 頂点を順にクリックし、最後に**ダブルクリックで確定**。
   - **円**: 中心をクリックし、もう一度クリック(またはドラッグ)して半径を決めます。
3. 作図した地物はファイルへ反映されます(約 0.3 秒後。`Ctrl+S` で保存)。

作図した地物を修正したいときは、**✏️ 編集**モードに切り替えてください。

---

## 図形を編集する(✏️ 編集モード)

**✏️ 編集**モードで地物を**クリックすると選択**され、赤くハイライトされて頂点に ● ハンドルが表示されます。編集できるのは選択中の 1 つの地物だけです。選択すると、右側に**プロパティパネル**が開きます。

### マウス操作の一覧

| 操作 | 動作 |
| --- | --- |
| 地物をクリック | 選択(ハイライト＋頂点●) |
| 頂点●をドラッグ | その頂点を移動(線・面) |
| 円のふちをドラッグ | 円の半径を変更(リサイズ) |
| **Shift＋ドラッグ** | 地物全体を平行移動(点・線・面・円すべて) |
| **Alt＋クリック**(頂点上) | その頂点を削除 |
| 何もない所をドラッグ | 地図のパン(移動しない) |
| 空白をクリック / **Esc** | 選択解除(編集終了) |

ポイント:

- **移動は必ず Shift＋ドラッグ**です。通常のドラッグでは移動しません(誤操作防止)。点も同様に、Shift＋ドラッグでのみ移動します。
- **頂点の削除は Alt＋クリック**。線は 2 点以上、面のリングは 3 点以上を維持します(それ以下にはできません)。

### コピー＆ペースト

- 選択中に **Ctrl+C**(Mac は Cmd+C)でコピー、**Ctrl+V** で少しずらして複製します。
- 同一エディタ内での複製で、円もそのまま複製できます。

### コピペや移動のショートカット早見表

| キー | 動作 |
| --- | --- |
| `Esc` | 選択解除(未確定の編集は破棄) |
| `Ctrl/Cmd + C` | 選択地物をコピー |
| `Ctrl/Cmd + V` | 複製を貼り付け |
| `Ctrl/Cmd + S` | 保存 |

## プロパティパネル(座標・属性の編集)

**✏️ 編集**モードで地物を選択すると、右側にパネルが開きます。ここでは**座標(緯度経度)**と**プロパティ(属性)**を編集できます。

### 「更新」ボタン方式(重要)

選択中の編集は、**すべて下書き**として扱われます。**地図上でのドラッグ/頂点編集も、パネルの座標・プロパティ入力も**、「更新」を押すまで**ファイルには保存されません**。

- **パネルの入力は Enter またはフォーカスアウトで地図の図形へ即反映**されます(保存は「更新」時)。地図に反映されるので、その場で位置を確認しながら微調整できます。
- **更新**: 下書き(地図＋パネルの編集)をまとめて確定し、ファイルへ保存します。
- **取消**: 下書きを破棄し、選択したときの状態へ戻します。
- 次の操作でも下書きは**破棄**されます: `Esc` / 空白クリック / 別の地物を選択 / ツールの切り替え。
- パネル下部に **「未確定の変更」/「同期済み」** の状態が表示されます。座標に不正な値(空欄など)があるときは「更新」は押せません。

> ねらい: 「地図でおおまかに配置 → パネルで数値を微調整(その場で地図に反映)→ 更新で保存」という流れです。保存するまでファイルに書き込まれないので、途中経過で差分や Undo 履歴が汚れません。

### 座標(緯度経度)の編集

パネル上部の「座標」で、選択地物の座標を数値で編集できます(緯度・経度の順)。**入力を Enter またはフォーカスアウトで確定すると、地図の図形へ即反映**されます(保存は「更新」時)。地図でドラッグ/頂点編集すると、パネルの値は地図の値で上書きされます(地図優先)。

- **入力欄にフォーカスすると、対応する地図上の頂点がハイライト**されます(どの頂点を編集しているか分かります)。円は中心が対象です。

| ジオメトリ | 表示・編集内容 |
| --- | --- |
| 点 (Point) | 緯度・経度 |
| 円 (Circle) | 中心の緯度・経度 ＋ 半径 (m) |
| 線 (LineString) | 頂点の表(各行を編集 / ＋で頂点追加 / ×で削除。最小 2 点) |
| 面 (Polygon) | リングごとの頂点の表(外周・穴。編集 / 追加 / 削除。最小 3 点) |

### name / color(組み込みフィールド)

`name` と `color` は、ワークスペース設定が無くても**常にパネルに表示**される組み込みフィールドです(新規作成した GeoJSON でもそのまま使えます)。

- **name**: 設定すると、その名称が**地図上にラベル表示**されます(点＝点の上、線＝中間、面・円＝内部)。編集中(選択中)も表示されます。
- **color**: カラーピッカーで色を選ぶと、地物が**その色(HEX)で着色**されます(点＝マーカー、線＝線色、面・円＝枠線＋半透明の塗り)。「クリア」で既定色に戻ります。
  - 着色は**選択を解除したとき**に反映されます(選択中は編集用の赤ハイライトが優先)。
- 同じキー(`name` / `color`)を設定で独自定義した場合は、その定義が優先されます。

### プロパティ(属性)の編集

ワークスペース設定で定義したフィールドが、パネルに入力欄として並びます(下記「設定」を参照)。

- テキスト・数値は **Enter またはフォーカスアウト**で、チェックボックス・選択肢は選んだ時点で、選択地物の `properties` に反映されます(保存は「更新」時)。
- **空欄にするとそのキーは削除**されます。
- パネルの「フィールド設定を開く」ボタンから、フィールド定義の設定画面へ移動できます。

## 設定(プロパティのフィールド定義)

プロパティとして編集できる項目は、ワークスペースの `.vscode/settings.json` に定義します。コマンド `GeoJSON Edit: フィールド設定を開く` からも編集できます。

```json
"geojson-edit.fields": [
  { "key": "name", "type": "string", "label": "名称" },
  { "key": "category", "type": "enum", "label": "分類", "options": ["都市", "経路", "エリア"] },
  { "key": "population", "type": "number", "label": "人口" },
  { "key": "visited", "type": "boolean", "label": "訪問済み" }
]
```

各フィールドのプロパティ:

| キー | 必須 | 説明 |
| --- | --- | --- |
| `key` | ✅ | `properties` に書き込むキー名 |
| `type` | ✅ | 入力の種類: `string` / `number` / `boolean` / `enum` / `color`(HEXピッカー) |
| `label` | | パネルでの表示名(省略時は `key` を表示) |
| `options` | | `type` が `enum` のときの選択肢(文字列の配列) |

- 設定は**リソーススコープ**です。フォルダ(ワークスペース)ごとに `.vscode/settings.json` で切り替えられます。
- フィールド定義が空のときは、パネルに「フィールド設定を開く」ボタンが表示されます。

## 円の保存形式

GeoJSON には Circle 型がありません。本拡張では円を **中心の Point ＋ `properties.radius`(メートル)** として保存し、再編集時に円へ復元します。

```json
{
  "type": "Feature",
  "geometry": { "type": "Point", "coordinates": [139.767, 35.681] },
  "properties": { "radius": 500 }
}
```

- `radius` はメートル単位です。地図は Web Mercator のため、中心緯度に応じて内部で投影半径へ換算しています。
- テキストで開き直すと、この形で保存されているのを確認できます。

---

## 座標系

- 保存: WGS84 (EPSG:4326) — GeoJSON 標準の緯度経度
- 表示: Web Mercator (EPSG:3857)

feature の `id` や top-level の独自メンバー(`crs`、`name` など)は、編集しても保持したまま書き戻します。
